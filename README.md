# JecnaGames

Školní projekt pro SPŠE Ječná - denní programátorské výzvy inspirované NYT Games.

## Hry
- Wordle Jecna - hádej IT pojem
- Connections - spoj pojmy do skupin  
- Fix Code - najdi chybu v kódu
- Cross Route - programátorská křížovka

## Stack
- Frontend: React + Vite + Tailwind
- Backend: PHP
- Databáze: MariaDB
